# vue3-transitions

根据 `vue-cron` 来改写的成 `vue3-corn` + `arco design`
## 参照

[vue-cron](https://github.com/1615450788/vue-cron)

## 安装

```
npm i vue3-corn
yarn add vue3-corn
pnpm add vue3-corn
```

## 使用

```
import { FadeTransition } from 'vue3-corn';
import 'vue3-corn/es/style/style.css';

<vue3-corn></vue3-corn>
```
